# Vim netrw + vinegar features
- `%` create a new file in current netrw folder
- `D` delete file
- `I` show help
- `gh` toggle display dotfiles
- `.` pre-populate file onto `:` command line
- `y.` yank an absolute path of file
- `R` rename file
- `i` toggle through different display
